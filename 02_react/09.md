<h1>Projeto Blog Pessoal - Criar os Componentes Navbar e Footer</h1>

Vamos apagar os componentes de Exemplo: **Card, Contador, Task** e vamos criar dois Componentes para o projeto Blog Pessoal: o **Navbar** e o **Footer**. Esses dois componentes serão inseridos na nossa página inicial e além disso, vamos aperfeiçoar o nosso componente **Home**, estilizando com o **Tailwind**.

<br />

<h2>👣 Passo 01 - Organizando a pasta Components</h2>

Antes de criarmos os Componentes **Navbar e Footer**, vamos apagar os componentes **Card, Contador e Task.**

1. Na pasta **components**, selecione os componentes descritos e os apague.

```react
import { FacebookLogo, InstagramLogo, LinkedinLogo } from '@phosphor-icons/react'

function Footer() {

    let data = new Date().getFullYear()

    return (
        <>
            <div className="
                flex 
                justify-center
                bg-indigo-900
                text-white
                ">
                <div className="
                    container 
                    flex 
                    flex-col 
                    items-center 
                    py-4
                    ">
                    <p className='
                        text-xl 
                        font-bold
                        '>
                            Blog pessoal Generation | Copyright: {data}
                        </p>
                    <p className='text-lg'>Acesse nossas redes sociais</p>
                    <div className='flex gap-2'>
                        <LinkedinLogo size={48} weight='bold' />
                        <InstagramLogo size={48} weight='bold' />
                        <FacebookLogo size={48} weight='bold' />
                    </div>
                </div>
            </div>
        </>
    )
}

export default Footer
```

<h2><img src="https://i.imgur.com/H9wEgsJ.png" title="source: imgur.com" width="4%"/>Passo 05 - Atualizar o Componente App</h2>



Vamos atualizar o Componente **App** de modo a exibir o Componente **Navbar** antes do Componente **Home** e o Componente **Footer** depois do Componente **Home**:

1. Abra o arquivo **App.tsx**, localizado na pasta **src**
2. Substitua o conteúdo do arquivo **App.tsx**, pelo código abaixo:

```react
import Footer from './components/footer/Footer';
import Navbar from './components/navbar/Navbar';
import Home from './pages/home/Home';

function App() {
  return (
    <>
      <Navbar />
      <Home />
      <Footer />
    </>
  );
}

export default App;
```
